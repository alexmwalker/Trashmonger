Inject-a-grid

Attach this CSS to any site and adjust the --columns and --lineheight variables to reveal any column grid and vertical rhythm. Handy for use with Stylebot or Stylus.

The grid is attached to the <body> currently. You might find that you need to attach it to the website container DIV or whatever...

The rows you see are half the <p> height. Two rows = one paragraph line-height, which makes it easy to use half-rows for vertical margins. Avoid setting explicit heights on anything. Only use vertical padding on the containers, not directly on text.
